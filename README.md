# Crer & Ser - Site Institucional

Site institucional para a Clínica Crer & Ser, desenvolvido com Next.js 14+ (App Router), TypeScript e Tailwind CSS.

## Design System

### Paleta de Cores (Terrosos & Aurora)
- **Background Base**: #FDFCF8 (Off-white suave)
- **Primary**: #A67B5B (Terracota suave)
- **Secondary**: #D4A373 (Bege Areia)
- **Accent**: #E6C8CA (Rosa Queimado) e #D8BFD8 (Lilás pálido)
- **Text**: #4A403A (Marrom café profundo)

### Tipografia
- **Headings**: Cormorant Garamond / Playfair Display (serif)
- **Body**: Inter / Lato (sans-serif)

### Efeito Organic Glassmorphism
Classes utilitárias disponíveis:
- `.glass` - Efeito glass padrão
- `.glass-strong` - Efeito glass mais intenso
- `.glass-subtle` - Efeito glass sutil

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Como executar

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar produção
npm start
```
